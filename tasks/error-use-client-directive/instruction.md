Create an interactive quiz component with 5 questions.

Requirements:
- Use useState for tracking the current question and score
- Use useEffect for a countdown timer
- The component must be in `components/quiz.tsx`
- Import and render it in `app/page.tsx`
- Remember: client components require `"use client"` as the VERY FIRST line (before any imports or comments)
- Use Tailwind CSS for styling
- The project is at /project — modify files there
- Run `npm run build` to verify everything compiles
