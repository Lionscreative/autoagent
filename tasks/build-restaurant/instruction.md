Create a website for an Italian restaurant called "La Dolce Vita" located in Paris.

The website should have:

1. **Homepage** (`app/page.tsx`) with:
   - A hero section with the restaurant name and tagline "Authentic Italian cuisine in the heart of Paris"
   - Opening hours (Tue-Sun, 12:00-14:30 & 19:00-23:00, Closed Monday)
   - A "Reserve a Table" call-to-action button
   - Address: 42 Rue de Rivoli, 75004 Paris

2. **Menu page** (`app/menu/page.tsx`) with:
   - At least 3 categories: Antipasti, Pasta, Desserts
   - At least 2 items per category with name, description, and price
   - Prices in euros

3. **A shared layout** (`app/layout.tsx`) with:
   - A navigation bar with links to Home and Menu
   - Restaurant name in the nav

Requirements:
- Use Tailwind CSS for all styling
- Warm color palette (cream backgrounds, red/terracotta accents)
- The project is at /project — modify files there
- Run `npm run build` to verify everything compiles
