Create a testimonials section with 5 customer quotes.

Include quotes like:
- 'Best service I've ever had — it's truly amazing!'
- 'They said "wow" when they saw the results'
- Quotes with both apostrophes and double quotes

Each testimonial should be in a grid layout with the customer name and role below the quote.

Requirements:
- Use Tailwind CSS
- Make sure all quotes are properly escaped (no build errors)
- The project is at /project — modify files there
- Run `npm run build` to verify everything compiles
