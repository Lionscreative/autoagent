Build a modern SaaS landing page for "CloudSync" — a file synchronization tool.

The landing page should be a single page (`app/page.tsx`) with these sections:

1. **Hero** — Headline: "Sync your files everywhere, instantly"
   Subheadline: "CloudSync keeps your team's files in perfect harmony across all devices"
   Two CTA buttons: "Start Free Trial" (primary) and "Watch Demo" (secondary/outline)

2. **Features** (3 features in a grid):
   - "Real-time Sync" — files update across devices in milliseconds
   - "End-to-end Encryption" — your data is always secure
   - "Team Collaboration" — share and edit files together seamlessly

3. **Pricing** (3 tiers in a grid):
   - Free: $0/mo, 5GB storage, 1 user
   - Pro: $12/mo, 100GB storage, 5 users, priority support
   - Enterprise: $49/mo, unlimited storage, unlimited users, dedicated support
   The Pro tier should be visually highlighted as "Most Popular"

4. **Footer** with:
   - Company name "CloudSync"
   - Copyright 2026
   - Links: Privacy, Terms, Contact

Requirements:
- Mobile-first responsive design using Tailwind breakpoints (sm:, md:, lg:)
- The hero should be full viewport height on desktop
- Features grid: 1 column on mobile, 3 on desktop
- Pricing grid: 1 column on mobile, 3 on desktop
- Use a modern color scheme (blue/indigo primary)
- The project is at /project
- Must pass `npm run build`
