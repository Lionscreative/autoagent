Create a SaaS pricing page with 3 tier cards (Starter, Pro, Enterprise).

Requirements:
- Each card should have a featured badge (e.g. "Most Popular", "Best Value")
- Use the shadcn/ui Badge component for the badges
- Include a CTA button on each card
- Each tier needs a price, feature list, and descriptive subtitle
- Use Tailwind CSS for styling
- The project is at /project — modify files there
- Run `npm run build` to verify everything compiles

Important: make sure any imported component actually exists in the project before using it. If a shadcn component is missing, create it or use a plain HTML/Tailwind alternative.
